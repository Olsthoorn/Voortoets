Cross section analysis¶

A cross section can often be analyzed (and often should be) on a larger model scale so see whether the numerical model makes sense. The simplest way is to take a one-layer cross section between boundaries with fixed head fed by recharge and compare the course of the head with that of the numerical model.

$$ h = h_L + \frac {h_R - h_L}{L}\,x + \frac N {2kD}\,(Lx - x^2)$$

import numpy as np
import matplotlib.pyplot as plt
import etc

def hx(hL=0., hR=10., N=0.002, kD=2000, L=20000):
    x = np.linspace(0, L, 2000)    
    h = hL + (hR - hL) / L * x + N / (2 * kD) * (L * x - x ** 2)
    return x, h
